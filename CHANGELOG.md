# 更新记录

## v0.3.6 - 2026-06-24

- 取消点赞改为即时切换未点赞状态，再由后台同步真实累计数。
- 修正鼠标停留时取消点赞仍显示填充色的问题。
- 移除长按蓄力圆环，改为点赞图标左右抖动提示。

## v0.3.5 - 2026-06-24

- 点赞区改为图标在上、文案在下的居中结构，融入周刊底部版式。
- 轻点点赞触发小范围礼花；长按约 0.9 秒显示蓄力环，完成后触发大面积礼花雨。
- 已点赞时再次轻点可取消当前期点赞，真实累计数同步减少。

## v0.3.4 - 2026-06-24

- 使用 SEXD 品牌 Logo 增加浏览器标签页 favicon。
- 增加 32px、192px 与 Apple Touch Icon 尺寸，并设置品牌主题色。

## v0.3.3 - 2026-06-24

- 点赞文案改为“周刊累计收到 xxx 个赞”，汇总所有期数的基础点赞与真实点赞。
- 使用 Cloudflare D1 保存真实点赞，每个匿名设备每一期只能点赞一次，点赞后不可取消。
- 切换期数时分别显示当前设备是否已为该期点赞，累计数字保持全刊共享。

## v0.3.2 - 2026-06-24

- 将订阅入口改为当前页面内弹出的飞书表单，读者无需跳转新标签页。
- 飞书自动化在新增邮箱后写入“已订阅”状态与提交时间。
- 完成桌面端和手机端的订阅弹窗适配。

## v0.3.1 - 2026-06-23

- 将底部订阅入口改为直达飞书多维表格表单，订阅邮箱由飞书表单直接写入订阅用户表。
- 保留 Cloudflare Pages Functions 代码作为后续 Open API 方案备用。

## v0.3.0 - 2026-06-22

- 增加飞书订阅表字段与 Cloudflare 环境变量配置说明。
